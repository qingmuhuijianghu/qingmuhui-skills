# 滋补品电商合规审查指南

## 适用平台

淘宝、天猫、京东、拼多多、抖音小店等国内电商平台。

---

## 绝对禁止表述

### 1. 医疗功效类（最严重）

| 违规词 | 风险等级 | 替代方案 |
|---------|---------|-----------|
| 抗肿瘤、抗癌、防癌 | ⛔ 严重 | 删除，或改为"传统食材" |
| 治疗、治愈、根治 | ⛔ 严重 | "日常调理"、"温和养护" |
| 消炎、抗菌、抗病毒 | ⚠️ 高风险 | "传统使用方式" |
| 降血脂、降血压、降血糖 | ⚠️ 高风险 | "日常养生"、"注重身体平衡" |
| 延缓衰老、抗衰老 | ⚠️ 中风险 | "抗氧化"、"传统智慧" |
| 排毒、清宿便 | ⚠️ 中风险 | 删除或改为"日常养护" |

### 2. 绝对化承诺类

| 违规词 | 风险等级 | 替代方案 |
|---------|---------|-----------|
| 假一赔十、假一赔万 | ⛔ 严重 | "品种不符承诺赔付" |
| 无效退款 | ⚠️ 高风险 | 删除 |
| 100% 有效、包治 | ⛔ 严重 | 删除 |
| 最强、第一、唯一 | ⚠️ 中风险 | 删除或改为客观描述 |

### 3. 权威性暗示类

| 违规表述 | 风险等级 | 说明 |
|---------|---------|------|
| "药典记载"（未标明具体出处） | ⚠️ 中风险 | 需标明具体典籍名称 |
| "官方推荐"、"专家认证" | ⛔ 严重 | 禁止无依据宣称 |
| 伪造证书、检测报告 | ⛔ 严重 | 涉及虚假宣传 |

## 合规表述模版

### 功效描述（合规版）

```
✅ 传统食材，注重日常养生人群的温和之选
✅ 日常守护，换季也不容易感到疲惫
✅ 应酬多、熬夜党的日常陪伴，给肝脏减减压
✅ 大自然的抗衰老力量，让身体保持好状态
✅ 日常调理，让身体的运转更顺畅
```

### 适用人群（合规版）

```
✅ 注重日常结节调养的人群
✅ 免疫力偏弱、换季易感人群
✅ 经常熬夜、饮酒应酬人群
✅ 注重血糖血脂管理的人群
✅ 追求健康养生的中老年人群
```

### 保障承诺（合规版）

```
✅ 只做桑树桑黄，品种不符承诺赔付
✅ 鉴别方法原原本本教给你
✅ 同批次实物拍摄，所见即所得
✅ 每块发出前均经五步核验
✅ 不影响二次销售，7天可退换
```

## 古籍引用规范

✅ **可以做的：**
- 引用具体典籍名称（《神农本草经》、《本草纲目》等）
- 引用原文（加引号）
- 注明朝代和作者

❌ **不可以做的：**
- 暗示"治疗疾病"
- 夸大古籍记载的功效
- 伪造古籍内容

**合规示例：**
```
《神农本草经》记载："桑耳，主女子漏下赤白汁，血病癥瘕积聚。"
```
**注意：** 引用后不加"所以能治疗 XX 病"等现代医疗解读。

## 图片合规

| 检查项 | 要求 |
|---------|------|
| 水印 | 去除 AI 生成水印（即梦、Midjourney 等） |
| 对比图 | 不得使用前/后对比宣称疗效 |
| 检测报告 | 只展示真实报告，不得 P 图 |
| 实物图 | 注明"实物拍摄"，不过度美化 |

---

## 审查流程

每次修改文案后，按以下清单检查：

- [ ] 无医疗功效词（抗肿瘤、治疗、消炎等）
- [ ] 无绝对化承诺（假一赔十、无效退款等）
- [ ] 无最高级形容词（最强、第一、唯一）
- [ ] 古籍引用有具体出处，未夸大解读
- [ ] 图片无水印、无虚假对比
- [ ] 功效描述使用"传统食材"、"日常养护"等合规措辞

---

## 平台违规后果

| 违规类型 | 平台处罚 |
|---------|---------|
| 医疗宣称 | 下架 + 扣分 + 罚款 |
| 绝对化承诺 | 警告 + 限期整改 |
| 虚假宣传 | 封店风险 |
| 图片违规 | 删除图片 + 扣分 |
