## 💡 5大省Token核心技巧

### 技巧1：精准定位（节省30-50%）
```
❌ 高消耗：帮我看看项目里有什么问题
✅ 低消耗：检查 D:\workbuddy\temp\xxx.html 第50行
```
**原理**：减少AI搜索文件的时间，直接给路径+行号

### 技巧2：一次说清（节省40-60%）
```
❌ 高消耗：帮我写篇文章...不对，这个词不对...再改一下...
✅ 低消耗：写公众号文章，主题AI工具推荐，秋叶风，800字，带3个小标题
```
**原理**：需求+风格+格式一步到位，减少来回修改轮次

### 技巧3：控制输出（节省20-40%）
```
❌ 高消耗：详细解释一下为什么要这样做
✅ 低消耗：只输出代码，不用解释
✅ 低消耗：直接给我结果，一句话总结
```
**原理**：明确告诉AI你想要什么格式，不要解释性废话

### 技巧4：批量创作（节省50-70%）
```
❌ 高消耗：帮我写一条小红书...再写一条...再来一条...
✅ 低消耗：帮我写5条小红书，都是关于AI工具推荐的
```
**原理**：一次提出多个需求，减少总对话轮次

### 技巧5：及时存档（节省50-80%）
```
每完成一个任务后说"做完了"，省Token助手帮你：
1. 把关键信息写入MEMORY.md（记忆文件）
2. 清理当前对话上下文
3. 继续下一个任务时无缝衔接
```
**原理**：避免上下文累积，减少每轮的Token消耗
