> 让每一次对话都花得更值

## 一句话说明

帮助你在WorkBuddy中节省Token消耗，提供任务模板、优化提问、上下文管理、批量创作指导。

## 下载方式

```
将「省Token助手」文件夹放入你的AI Skills目录：

WorkBuddy:  ~/.workbuddy/skills/
QClaw:      ~/.qclaw/skills/
```

## 使用方法

### 1. 问省Token方法
```
你：怎么省Token
AI：给你5大技巧 + 你的任务类型专属模板
```

### 2. 一次说清模板
```
写公众号文章
主题：AI工具推荐
风格：秋叶风
字数：800字
要求：带小标题，文末有互动引导
```

### 3. 任务完成存档
```
你：做完了
AI：记录到记忆文件 + 询问是否清理上下文
```

### 4. 继续旧任务
```
你：继续做XX项目
AI：读取记忆文件恢复上下文，无缝衔接
```

### 5. 批量创作
```
你：帮我写10条小红书图文
AI：一次生成10条，不要一条一条来
```

## 5大核心技巧

| 技巧 | 示例 | 节省 |
|------|------|------|
| 精准定位 | "检查D盘test.html第50行" | 30-50% |
| 一次说清 | 需求+风格+格式一步到位 | 40-60% |
| 控制输出 | "只输出代码，不用解释" | 20-40% |
| 批量创作 | 一次10条，不要一条一条 | 50-70% |
| 及时存档 | 任务完成即存档+清理 | 50-80% |
