## 模式B：沙龙活动SOP

### 晓云姐旗下5大沙龙类型

#### 1. 问题肌管理体验课
- **目标**：解决皮肤问题 + 建立专业信任
- **人数**：5-15人（小班精准）
- **时长**：2-3小时
- **流程**：
  1. 皮肤检测/问诊（20分钟）—— 用专业度镇场
  2. 问题成因讲解（30分钟）—— 痤疮/敏感肌/色斑分类
  3. 净彤颜产品体验（60分钟）—— 手法+产品演示
  4. 个人方案定制（30分钟）—— 一对一方案
  5. 互动答疑+后期跟进安排（20分钟）

#### 2. 抗糖减脂轻食营
- **目标**：体重管理 + 亚健康调理 + 康婷产品植入
- **核心原则**：一人一方案，因人而异，绝不千篇一律
- **时长**：7天/21天打卡营
- **流程**：
  1. 入营测评（基础代谢/身体成分/体质分析）
  2. 一对一定制方案（根据体质、目标、生活习惯量身设计）
  3. 康婷相关产品配合说明
  4. 打卡群互动 + 阶段成果展示
  5. 结营分享会（引向深度合作）

#### 3. 高密度抗衰沙龙
- **目标**：高净值客户 + 抗衰理念植入 + 康婷护肤产品体验
- **核心卖点**：纯手工技术，不依赖仪器，有温度，30+冻龄40+逆龄
- **收费**：纯手工服务费（产品由顾客自行通过康婷会员购买）
- **四大功效**：
  1. 改善皮肤粗糙、毛孔堵塞
  2. 改善松弛下垂：抬头纹/川字纹/法令纹/颈纹/眼角纹
  3. 改善缺水引起的脱皮、卡粉
  4. 改善皮肤发黄和过敏，调理受损肌肤
- **使用产品**：根据肤质定制搭配康婷护肤系列
- **手法体系**：独家纯手工手法，到店沉浸式体验，不依赖仪器
- **为什么推康婷产品**：自主研发自主生产自主检测，可参观生产车间，康汇医院为会员健康保驾护航

#### 4. 五行养生茶道疗愈沙龙
- **目标**：品质生活方式切入 + 建立情感连接 + 身心疗愈体验
- **核心价值**：以茶入道，唤醒内在记忆，美·静·健康三合一
- **茶道本质**：不是茶艺（品味道看姿势），而是茶道——以茶养生，滋养五脏
- **五行对应**：木（肝）· 火（心）· 土（脾）· 金（肺）· 水（肾），配合五音共振
- **七碗茶仪式**：第1杯和第7杯敬天地，第2-6杯对应五脏滋养
- **核心理念**：人有三宝精·气·神；五行养生茶生元精、补元气、养元神
- **茶的地位**：上药，百利而无一害，百药之王；以茶为师，茶走到哪里滋养哪里
- **五行相生**：肝木→心火→脾土→肺金→肾水→肝木（循环相生）
- **操作方式**：全程引导，沉浸式体验，到店感受茶道魅力
- **茶后特色手法**：独家手法体验，茶道与美业的融合
- **流程框架**：茶席布置 → 理念讲解 → 茶道体验 → 分享交流 → 健康话题自然过渡

#### 5. 读书会（整理生命·反转生命）
- **目标**：高质量用户筛选 + 理念共鸣 + 事业吸引
- **核心定位**：从源头改变，不是卖产品而是传递生命观
- **流程**：
  1. 主题导入（一本书/一个话题）
  2. 分享与共鸣（参与者轮流发言）
  3. 晓云姐分享故事（真实经历，感染力）
  4. 康婷大健康事业愿景自然切入
  5. 感兴趣者单独深聊

#### 6. 吃喝拉撒睡沙龙（不销而销主场）
- **目标**：讲温度、讲营养、讲生活习惯 → 产品自然融入
- **与其他沙龙的关系**：
  - 茶道沙龙 / 读书会 = 建信任，不销产品
  - 吃喝拉撒睡 = 建信任 + 可以植入产品，但仍然不销而销
- **沙龙宗旨**：讲真实，讲自己，讲温度；顾客已经被"推销"整怕了，有防备心，太明显的推销没人愿意参加

### 沙龙产品植入原则：不销而销（晓云姐实战心法）

**核心公式：真实体验讲自己 + 沙龙场景自然用 + 留入口不追 = 不销而销**

#### 三不原则
- **不销**：现场不出现任何产品推销话术
- **不强**：沙龙结束后不追着客户问要不要买
- **不急**：不催单、不群发、不私信轰炸

#### 三步落地法

**第一步：沙龙中——让产品"自己说话"**
- 把产品变成沙龙体验的一部分，不是"卖"而是"用给你看"
- 客户吃进嘴里、感受在皮肤上，不是看瓶子

**第二步：沙龙后——只留一句话**
- 沙龙结束不做总结推销，只说：
  > "我平时也在吃/用，有需要的姐妹可以微我"
- 留联系方式入口就够了，不追

**第三步：朋友圈——第二天发感受不发广告**
- 第二天朋友圈：发沙龙现场感受
- 不提产品名称，只发"今天XX沙龙很开心，学到了…"
- 评论区自然有人问，自然回答

#### 康婷产品沙龙植入对照表

| 沙龙类型 | 产品怎么进 | 晓云姐话术（讲自己，不销） |
|---------|-----------|------------------------|
| 吃喝拉撒睡 / 轻食营 | 现场用白芸豆L-阿拉伯糖做点心给客户吃 | "我自己每天早上喝一杯，喝了半个月脸色亮了" |
| 抗糖沙龙 | 客户现场吃到嘴里，不是看瓶子 | "这个是我老公的痛风吃好的，后来我自己也跟着吃了" |
| 茶道沙龙 | 茶是主角，结束后自然聊健康习惯 | "茶道之后我配合吃这个，整个人状态好了很多" |
| 高密度抗衰 | 手法服务中完成产品体验 | 客户感受到效果自己来问，不主动推荐 |

#### 事业引入三层次（信任够了自然过渡）
1. **温度** → 先讲人，晓云姐自己的故事，不是讲产品
2. **理念** → 康婷"健康·美丽·心能量"呼应沙龙里聊到的生命观
3. **事业** → 感兴趣的人主动来问："晓云姐，这个事业怎么做？"

> **口诀：沙龙是养鱼，不是钓鱼。鱼养好了，自然游过来。**

## 模式D：美容院盈利模型

### 核心矛盾（常见误区）
- ❌ 误区：低价全包产品 → 忽略手艺价值，等于白干
- ❌ 误区：让顾客走会员价 → 自己没利润差价了
- ✅ 正确理解：产品和手艺分开算，收益从公司体系来，不从顾客身上硬扣

### 康婷会员政策
- **首次**：享受会员折扣拿货
- **二次起**：享受更深折扣
- **本质**：会员自己下单、自己付款，钱进康婷公司，晓云姐不碰货款

### 三条收入线（晓云姐的真实盈利来源）

#### 收入线①：手艺服务费
- 纯手工服务费，当场收取
- 触发条件：无，只要做服务就有
- 跟顾客说："手法是我的独家，外面做不了"

#### 收入线②：直推奖（康婷公司一次性奖励）
- 顾客首次加入会员 → 康婷公司一次性奖励给推荐人
- 具体比例由康婷公司政策规定，让顾客自己去康婷了解
- 晓云姐的价值：把人带来、把信任建立好、把会员转化做掉

#### 收入线③：复购奖励（康婷公司持续奖励）
- 顾客二次复购累计达标 → 康婷公司一次性奖励给推荐人
- 之后每再达标继续有奖励
- **具体比例由康婷公司政策规定**，让顾客自己去康婷了解

### 跟顾客怎么讲（不销而销的自然过渡）
> "你今天做了一次抗衰体验，感觉怎么样？你要是想长期保养，我可以帮你加入康婷会员，享受会员折扣，比你单次买便宜太多了。公司还有奖励政策，你自己去了解了解，感觉合适我们再聊。"

### 晓云姐不需要做的事
- ❌ 不需要自己压货
- ❌ 不需要在店里卖零售价
- ❌ 不需要跟顾客解释具体奖励比例
- ✅ 只需要：做好手艺 → 建立信任 → 自然转化会员 → 康婷公司发奖励

### 话术要点（敏感信息处理原则）
- 康婷产品零售价/会员政策：**可以讲**（公开信息）
- 康婷具体奖励比例：**不主动透露**，引导顾客自己去康婷了解，显得客观可信
- 手艺费：**直接报价**，不含产品，明明白白

> **口诀：沙龙是养鱼，不是钓鱼。鱼养好了，自然游过来。**

## 模式C：客户承接话术

### 新客户第一次来沙龙（问题肌切入）

```
欢迎话术（温暖专业）：
"欢迎来！先别急着说哪里不好，我想先看看你的皮肤状态，
皮肤会说话，我们听它说完，再找解决方案。"

皮肤问诊三问：
1. 你的皮肤问题困扰你多久了？
2. 用过什么方法，效果怎么样？
3. 你最想先解决哪个问题？

建立信任的关键话术：
"皮肤问题不是表面的事，它跟你的内在状态有关——
睡眠、饮食、情绪，都会写在脸上。
我们这里解决的不只是皮肤，是整个人的状态。"
```

### 从皮肤问题引入健康话题（自然过渡）

```
"你皮肤爱出油/长痘，其实跟血糖波动有很大关系。
高糖饮食会加速皮脂分泌，我们有个抗糖饮食方案，
配合护理效果会快很多，你想了解一下吗？"
```

### 从健康话题引入康婷事业（水到渠成）

```
"其实我做这个不只是帮人解决皮肤问题，
我背后有一个康婷的大健康体系，
从皮肤到身体到心灵，都有完整的方案。
很多来这里的姐妹，后来也成了我一起做事业的伙伴。"
```

## 模式E：内容文案生产

### 5类高频文案模板

**1. 效果见证类（转化力最强）**
```
客户叫[昵称]，[肤质/问题]困扰了她[多久]
[之前用过什么/结果怎样]
来了[X次]之后——[具体变化]
她说最大的改变不只是皮肤，是[情绪/状态/自信]
这才是我做这件事的意义。
```

**2. 干货科普类（引流力最强）**
```
[问题现象]（比如：为什么你的痘痘总是在[部位]反复？）
原因：[1-2个核心原因，专业但好懂]
解法：[具体可操作的1-2个步骤]
（评论区"想了解"，我发你详细方案）
```

**3. 活动招募类（社群用）**
```
[沙龙名称]正在招募 | [日期/地点]
适合你，如果你——
✓ [痛点1]
✓ [痛点2]
✓ [痛点3]
名额只有[X]个，已报名[X]个姐妹
回复"报名"或联系晓云姐 →
```

**4. 生活方式类（品牌感建设）**
```
今天[茶道/读书/沙龙现场]的一张照片
[1-2句感受，不励志不鸡汤，真实就好]
生活可以很美，当你开始认真对待自己。
```

**5. 康婷事业吸引类（针对有想法的人）**
```
很多人问我，你怎么可以同时做[X件事]？
因为它们背后是同一件事：帮人变得更好。
皮肤、身体、事业，其实是一体的。
如果你也在找一件值得做的事，欢迎聊聊。
```
